How to use

🚀 Quick Start

1Open this page in Chrome. Tap ⋮ → Add to Home Screen to install as a PWA — this unlocks full auto-play (music launches automatically, no tap needed).
2Relaunch from the home screen icon. The green "Running as app" badge confirms auto-play is active.
3Tap ▶ Start Jog Session. Screen stays on (wakelock active), timer starts, motion detection arms.
4Start running. Ring turns green at jog cadence. After ~4 seconds sustained, a beep fires and YouTube Music opens automatically.
5Tap ■ Stop when done. Session saves to history automatically.

🎵 Auto-play: browser vs installed

In browser mode, Chrome blocks auto-launching apps — tap the green banner (one tap). In installed PWA mode (launched from home screen icon), Chrome grants autoplay permission and YouTube Music opens automatically the moment jogging is detected.

⚙️ Sensitivity

  • 1–2 Low: Strong impact needed — loose in pocket
  • 3 Medium: Default — armband or hand
  • 4–5 High: Very sensitive — may false-trigger walking

💡 Tips

Hold the phone or use an armband. Loose in a bag reduces step detection accuracy.

Ring colour: yellow = walking · green = jogging · red = sprinting

Set a custom cadence range in the main screen if you run slower or faster than the default 110–210 spm.
