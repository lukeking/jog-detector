How to use

🚀 Quick Start

1Open in Chrome. Tap ⋮ → Add to Home Screen to install as a PWA — unlocks full auto-play.
2Relaunch from the home screen icon. The green "Native app" badge confirms auto-play is active.
3Tap ▶ Start Jog Session. Screen stays on, timer starts, motion detection arms.
4Start running. Ring turns green at jog cadence. After ~4 seconds, a beep fires and YouTube Music starts automatically.
5Tap ■ Stop when done. Session saves to history.

🎵 Auto-play: browser vs installed

In browser mode, Chrome blocks auto-launching apps — tap the green banner. In APK / installed PWA mode, YouTube Music opens and starts playing automatically.

⚙️ Sensitivity

  • 1–2 Low: Strong impact — loose in pocket
  • 3 Medium: Default — armband or hand
  • 4–5 High: Very sensitive — may false-trigger walking

💡 Tips

Hold the phone or use an armband. Loose in a bag reduces accuracy.

Ring colour: yellow = walking · green = jogging · red = sprinting

Set a custom cadence range if you run slower or faster than 110–210 spm.
